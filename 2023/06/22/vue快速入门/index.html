    vue快速入门

    1、v-if和v-show的区别：

    v-if：不会在页面上渲染
    v-show： 会渲染在页面上，但是会通过style="display: none;"来隐藏起来

    用哪个？

    如果是需要经常改变的，用v-show，这样性能损失小一些

    2、vuex快速入门

    首先安装：npm install vuex --save
    然后新建src/store/index.js

    然后记得在main.js里面挂载

    使用：
    把所有需要共享的状态都放在state里面,比如 name: '张三', age: 21

    取值：
    有两种方法，1：直接使用 this.$store.state[属性] ，（this 可以省略）

    直接用this.$store.state.name是最简单的，如果要改变值直接this.$store.state.name = '李四'就可以了，或者this.$store.state[name] = '李四'
    2：通过 mapState把 store 映射到 组件的计算属性，就相当于组件内部有了 state 里的属性

    另外还有getter、module、action暂时不管，用不上

    3、vue router快速入门

    安装和使用路由：

    页面路由跳转：
    用this.$router.push('/home')把当前需要跳转的地址push进去就好了

    到这里，我们已经可以跳转页面了，但是还不够，肯定还需要携带参数
    动态路由：
    params：

    • 配置格式：/router/:id
    • 传递方式： 在path后面跟上对应的值
    • 传递后形成的路径： /router/123 或者 /router/abc

    query：

    • 配置格式： /router，也就是普通配置
    • 传递方式： 对象中使用query的key作为传递方式
    • 传递路径： /router?id=123 或者 /router?id=abc
